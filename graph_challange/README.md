## Approach & Efficiency

### big O

function add_vertex:
time: O(1)
space O(1)

function add_edge:
time: O(1)
space O(1)

function get_vertices:
time: O(n)
space O(1)

function get_neighbors:
time: O(n)
space O(1)

function size:
time: O(n)
space O(1)

function breadth_first:
time: O(n)
space O(n)

function business_trip:
time: O(v+n*e)
space O(v+n)
v = vertices
n = list size
e = edges

function depth_first:
time: O(V)
space O(V)
